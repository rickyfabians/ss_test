# Backend for sstest


## Getting Started


### Installing

* NodeJs
* npm

- Install dependencies
```
cd <project>
npm install
```
- Import db to mysql
```
<\Backend\db\salestock_db 20181010 2229>
```

## Running nodeJS

- Build and run the project
```
npm start
```
- If it is successful, the it will be launched on `http://localhost:4000` 

- Don't forget to start the mysql
